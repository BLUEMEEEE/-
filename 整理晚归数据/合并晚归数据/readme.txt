首先要有python环境，如果自己电脑上没有的，你们找下装有python的朋友跑一下就好了
还要装两个包xlrd和xlwt

把你要合到一起的所有的excel文件全部放到该目录下的input文件夹（必须保证每个excel文件下的从左到右学院的顺序是一样的），然后双击merge.py文件，
会输出一个output.xls文件，这就是结果

这个工具只会傻瓜式的把input文件夹里面的所用excel中，同是第一个sheet的所有记录合并到一起，并以此类推
并不会进行判断是不是一样的学院，所以用之前必须保证每个excel文件的sheet的学院顺序是一样的

最好不要放在c盘吧，我也不知道会不会缺失文件读取权限
